# Wellfit-html
Second semester Web development project

In this project, I learnt how to build a website for a gym management system. The website consists of 
 * homepage
 * timetables for the gym
 * nearest centres
 * Google map API which provide routes
 * image gallery
 * Price details
 * Franchise
